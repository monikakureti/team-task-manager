# Team Task Manager

Team Task Manager is a simple and collaborative task management web application built using the MERN stack.
The application helps teams create workspaces, manage tasks, and organize projects efficiently.

This project was developed to provide a clean and user-friendly platform where users can manage their daily workflow and collaborate within different workspaces.

---

# Features

* User Registration & Login
* Secure JWT Authentication
* Workspace Creation & Management
* Multiple Workspaces Support
* Task Creation and Tracking
* Dashboard Overview
* Role-based Workspace Access
* Responsive Modern UI

---

# Tech Stack

## Frontend

* React.js
* Tailwind CSS
* React Router DOM
* Axios

## Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication

# Installation

## Clone the Repository

```bash
git clone https://github.com/monikakureti/team-task-manager.git
```

---

# Backend Setup

```bash
cd backend
npm install
npm run dev
```

Create a `.env` file inside the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
